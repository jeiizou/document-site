# 数据结构-树


<!-- TODO -->

树是一种经典的数据结构. 特点是:

1. 每个节点有零或者多个子节点
2. 没有父节点的节点称为根节点
3. 每一个非根节点有且只有一个父节点
4. 每个子节点可以分为多个不相交的子树


## 树的分类

- 无序树: 树中任意节点的子节点之间没有顺序关系
- 有序树: 任意子节点有顺序关系
- 二叉树
- B树
- 红黑树
- 堆
- 伸展树
- 哈弗曼树

等等...

## 树的遍历

树的遍历有四种: 先序遍历, 中序遍历, 后序遍历, 层次遍历. 一个一个来看

### 先序遍历

先序遍历(Pre-order)，按照根左右的顺序沿一定路径经过路径上所有的结点。在二叉树中，先根后左再右.
